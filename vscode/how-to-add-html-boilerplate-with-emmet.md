# How to add an HTML boilerplate with Emmet

Place the cursor in your empty html file and type the charater `!` followed by `Tab` key press and voil√°!

This is what is added to your file:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```

Happy Coding!
